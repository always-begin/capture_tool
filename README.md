# CapTool : キャプトル (画面キャプチャ・切り出しツール)

シンプルで高機能なデスクトップ画面キャプチャツール。範囲選択・プレビュー・連続保存をひとつのウィンドウで完結させます。

## 概要

CapTool は Python / Tkinter ベースのデスクトップキャプチャツールです。  
半透明のオーバーレイ上でドラッグするだけで任意の矩形範囲を選択でき、**通常キャプチャ**と**連続保存**の2モードを切り替えて使用できます。

### CapTool の特徴

**リアルタイムキャプチャ**  
Snipping Tool は一度全画面をフリーズさせてから切り抜く「事後トリミング方式」ですが、CapTool はオーバーレイを表示したまま範囲を確定してから撮影するため、画面の内容はキャプチャ直前まで動き続けます。ゲームのスコア、ターミナルのログ、動画の特定フレームなど、「この瞬間を撮りたい」という場面で威力を発揮します。連続保存モードと組み合わせると、Space キーを叩くたびにリアルタイムで変化する画面を次々と記録できます。

**前回範囲の自動復元**  
設定ファイルに選択範囲を保存し、次回起動時に自動で同じ範囲を復元します。同じ領域を繰り返しキャプチャする作業（ゲームのスコア記録、定点監視など）で手間を大幅に削減できます。

**キーボード・ホイールによるピクセル単位の微調整**  
マウスドラッグでは難しい1px単位の精密な範囲調整を、矢印キーとマウスホイールで実現します。

**連続保存モード**  
スペースキーを押すたびに `capture_YYYYMMDD_HHMMSS_ffffff.png` という形式でタイムスタンプ付きのファイルを自動保存します。録画の代わりとして定期的にスクリーンショットを記録したい場合などに便利です。

**軽量・依存ライブラリが少ない**  
GUIフレームワークとして CustomTkinter を採用し、インストール後すぐに起動できます。Electron ベースのツールのような大きなランタイムは不要です。

## 動作環境

- Python 3.9 以上
- Windows 10/11（クリップボードコピー機能はWindowsのみ）
- macOS / Linux（クリップボードコピーを除く全機能が動作）
- マルチモニター環境に対応

## インストール

### Windows（推奨）

GitHub の [Releases](https://github.com/yourname/captool/releases) ページから最新の `CapTool.exe` をダウンロードして実行するだけです。Python や追加ライブラリのインストールは不要です。

> 実行ファイルは GitHub Actions（CI）によって自動ビルドされ、リリースごとに公開されます。

#### ソースから実行する場合

```bash
git clone https://github.com/yourname/captool.git
cd captool
pip install -r requirements.txt
python main.py
```

`requirements.txt` の内容例:

```
customtkinter
Pillow
mss
screeninfo
pywin32  # Windowsのみ必要
```

exeをダウンロードした場合はそのままダブルクリックで起動できます。ソースから実行する場合:

## 操作方法

### メインウィンドウ

| 操作 | 説明 |
|---|---|
| **CAPTURE START** ボタン | キャプチャオーバーレイを開く |
| モードセレクター | 「通常モード」と「連続保存」を切り替え |
| **コピー** ボタン | キャプチャ画像をクリップボードにコピー（Windowsのみ） |
| **画像を保存** ボタン | PNG形式でファイルに保存 |

#### プレビューエリアの操作（通常モード）

| 操作 | 説明 |
|---|---|
| マウスホイール | ズームイン / ズームアウト |
| 左ドラッグ | 画像のパン（移動） |
| 右クリック | ズームとオフセットをリセット |

---

### キャプチャオーバーレイ

オーバーレイを開くと、全画面が半透明のブラックで覆われます。前回の選択範囲がシアン（水色）の破線で自動復元されます。

#### マウス操作

| 操作 | エリア | 動作 |
|---|---|---|
| 左クリック | 全エリア | キャプチャを実行<br>**通常モード**: プレビューに表示してオーバーレイを閉じる<br>**連続保存**: ファイルに保存してオーバーレイを維持 |
| 左ドラッグ | 矩形の外 | 新しい矩形範囲を作成<br> ※最小範囲の制限あり|
| 左ドラッグ | 矩形の内側 | 既存の範囲を移動 |
| 右クリック | 全エリア | 選択範囲を削除 |
| 中クリック | 全エリア | オーバーレイを閉じてキャンセル |
| ホイール上下 | 全エリア | 矩形の**高さ**を10pxずつ変更 |
| `Shift` + ホイール | 全エリア | 矩形の**幅**を10pxずつ変更 |
| `Ctrl` + ホイール | 全エリア | 矩形の高さ／幅を**1pxずつ**変更（精密調整）

#### キーボードショートカット

| キー | 説明 |
|---|---|
| `Space` | キャプチャを実行 |
| `Esc` | オーバーレイを閉じてキャンセル |
| `↑` `↓` `←` `→` | 矩形の**右下端**を10pxずつ移動（サイズ変更） |
| `Ctrl` + 矢印キー | 矩形の右下端を**1pxずつ**移動（精密調整） |

---

### 連続保存モード

1. モードセレクターで「連続保存」を選択
2. **CAPTURE START** をクリックし、保存先フォルダを選択
3. オーバーレイが開いたら `Space` を押すたびにキャプチャが保存される
4. `Esc` でオーバーレイを閉じ、メインウィンドウに戻る
5. 「📁 保存先をエクスプローラーで開く」ボタンで保存先を確認

保存ファイル名の形式: `capture_YYYYMMDD_HHMMSS_ffffff.png`

## 設定ファイル

選択範囲（座標・サイズ）は以下の場所に自動保存されます。

| OS | パス |
|---|---|
| Windows | `%APPDATA%\CapTool\settings.json` |
| macOS | `~/Library/Application Support/CapTool/settings.json` |
| Linux | `~/.config/CapTool/settings.json` |

```json
{
  "x": 200,
  "y": 150,
  "w": 800,
  "h": 600
}
```

設定を初期化したい場合はこのファイルを削除してください。
