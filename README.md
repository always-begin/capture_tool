# CapTool (画面キャプチャ・切り出しツール)

マルチモニター環境に対応し、直感的な範囲選択と高度な微調整機能を備えた、クリエイター・開発者向けの画面キャプチャツールです。

## 📁 フォルダ構成

```text
capture_tool/
├── main.py                     # アプリ起動スクリプト
├── README.md                   # 本書（取扱説明書）
└── src/
    ├── ui/
    │   ├── main_window.py      # メインスタジオ画面 (CapToolStudio)
    │   └── overlay.py          # 範囲選択オーバーレイ画面 (CaptureOverlay)
    └── core/
        └── utils.py            # キャプチャ・クリップボード・設定保存の共通ロジック

```

---

## 🚀 基本的な使い方

1. **ツールの起動**
* `main.py` を実行すると、メインのスタジオ画面（操作パネル）が起動します。


2. **キャプチャモードの選択**
* **通常モード (`Single`)**: 選択した範囲を1回キャプチャし、クリップボードにコピーして終了します。
* **連続モード (`Multi`)**: 選択した範囲を、キーを押すたびに指定フォルダへ連続で自動保存します。


3. **範囲選択の開始**
* 「キャプチャ開始」ボタンを押すと画面全体が選択モード（オーバーレイ）に切り替わります。

## 🎮 範囲選択画面（オーバーレイ）の操作方法

範囲選択画面では、マウスだけでなく、キーボードやマウスホイールを使って **1ピクセル単位の精密な位置・サイズ調整** が可能です。

### 1. マウス操作

| 操作 | 動作 |
| --- | --- |
| **ドラッグ（未選択領域）** | 新しいキャプチャ範囲を四角く指定します。 |
| **ドラッグ（選択範囲の内側）** | 現在のキャプチャ範囲のサイズを維持したまま、位置を移動します。 |
| **左クリック（ドラッグなし）** | **【確定】** 現在の範囲でキャプチャを実行します。 |
| **右クリック** | 現在の選択範囲をクリア（リセット）します。 |
| **ホイールクリック（中クリック）** | キャプチャをキャンセルしてメイン画面に戻ります。 |

### 2. キーボード操作

| キー | 動作 |
| --- | --- |
| **`Space` キー** | **【確定】** 現在の範囲でキャプチャを実行します。 |
| **`Escape` キー** | キャプチャをキャンセルしてメイン画面に戻ります。 |
| **`↑` `↓` `←` `→`（矢印キー）** | 選択範囲の**右下隅**を動かし、サイズを **10px 単位**で微調整します。 |
| **`Ctrl` + 矢印キー** | 選択範囲の**右下隅**を動かし、サイズを **1px 単位**で精密に微調整します。 |

### 3. マウスホイールによる微調整

選択範囲にカーソルを合わせた状態でホイールを回転させると、サイズを素早く調整できます。

| 操作 | 動作 |
| --- | --- |
| **マウスホイール回転** | 選択範囲の**高さ（縦幅）**を 10px 単位で調整します。 |
| **`Ctrl` + ホイール回転** | 選択範囲の**高さ（縦幅）**を 1px 単位で精密に調整します。 |
| **`Shift` + ホイール回転** | 選択範囲の**横幅**を 10px 単位で調整します。 |
| **`Ctrl` + `Shift` + ホイール回転** | 選択範囲の**横幅**を 1px 単位で精密に調整します。 |

## ✨ 便利な機能

* **マルチモニター完全対応**
* メインモニターだけでなく、左右や上下に配置されたサブモニターも含めた全画面領域を1つのキャンバスとしてシームレスに操作・キャプチャできます。


* **位置の自動記憶 (レジューム機能)**
* 一度キャプチャした範囲（座標とサイズ）は、OSの設定領域（Windowsなら `AppData`）に自動で保存されます。次回起動時や、連続モードで再度画面を開いた際、**前回の位置が最初から復元された状態**でスタートできます。


* **連続保存時のフラッシュエフェクト**
* 連続モードで `Space` キーやクリックでキャプチャを行うと、選択枠が一瞬「白く太く」光ります。裏で正しく保存されたことが視覚的に分かる安心設計です。
